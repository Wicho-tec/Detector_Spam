## ¿Qué es y qué hace la práctica?/¿Qué aprendimos?
El programa es un detecor de spam con una interfaz gráfica simple, funciona pasandole un texto de un correo y el programa detecta si es spam o no.
Aprendimos a diseñar de manera simple una GUI para nuestros proyectos en python.

![Imágen del programa en ejecución](image-1.png)

## ¿Cómo lo hicimos? / Tecnologías-Librerías utilizadas
Utilizamos diversas librerias, cada una con un objetivo en específico. Nuestro propio modulo donde teniamos las funciones del detector con las librerias para entrenar el modelo y en el archivo screen.py usamos tkinter para la GUI.

## Trabajo futuro / Oportunidades de mejora / Solución de sus posibles errores
Este mini proyecto nos puede ayudar en trabajos futuros principalmente en cuanto al uso de interfaces gráficas. Aunque podemos mejorar en aprender cómo hacer unas con diseño más complejo para que sean más atractivas.
